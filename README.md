# Haadia Personal Website

This is a Vite + React personal website with a subtle purple tech theme and 3D-style animation.

## Upload to GitHub without running locally

1. Unzip this folder.
2. Upload all files to your GitHub repository.
3. Go to the repository's **Actions** tab.
4. The included GitHub Pages workflow will build and deploy the site automatically.
5. Your site URL should be: https://haadia-a.github.io/haadiaamjad.github.io/

## Editing content

Most text is inside `src/App.jsx`.
Colors and layout are inside `src/styles.css`.
